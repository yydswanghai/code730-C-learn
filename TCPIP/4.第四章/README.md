## 理解 TCP 和 UDP

根据数据传输方式的不同，基于网络协议的套接字一般分为TCP套接字和UDP套接字。
因为TCP套接字是面向连接的，因此又称基于流（stream）的套接字。

TCP是Transmission Contorl Protocal（传输控制协议）的简写，意为“对数据传输过程的控制”。
因此，学习控制方法及范围有助于正确理解TCP套接字。
